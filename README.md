# hello-world-example
A Hello World Example Package

## Installation
Install via NPM:

```bash
npm install hello-world-example

```

## Usage

#### javascript

```javascript

var myApp = require("hello-world-example");

myApp.printMsg(); // print and return " Hello World! " 

```

#### TypeScript

```typescript

import * as myApp from 'hello-world-example';

myApp.printMsg(); // print and return " Hello World! " 


```
